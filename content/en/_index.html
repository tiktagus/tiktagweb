How TikTag works in command-line:

{{< card-code lang="Shell" >}} Host an asset, > tiktag myfilename.png > Tik ...Tag ... your asset is successfully hosted at, > https://s3.tikoly.com/village/563583552944996352.png {{< /card-code>}} {{< card-code lang="Shell" >}} Retrieve an asset's URI > tiktag search myfilename.png > We found this asset at, > https://s3.tikoly.com/village/563583552944996352.png {{< /card-code>}} {{< card-code lang="Shell" >}} Verify an asset's ownership, > tiktag whose myfilename.png > We found, > Sui Wallet https://etherscan.io/address/0x6ce19a00f0424b27bc2fce5ec3f10ba022a42370 > Please visit https://tiktag.us/ for more options. {{< /card-code>}}

{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} `TikTag` is a tool to make the handling of digital asset, such as photo, **auditable**.
Tik...you host your image somewhere of your choosing.
Tag...you get the image's URL for authorized use or public sharing.
